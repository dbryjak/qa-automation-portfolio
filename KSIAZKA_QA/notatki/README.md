# Notatki dzienne — Książka QA

Tu zapisujesz postęp **na koniec dnia** albo **w trakcie pracy** (gdy coś ważnego się wydarzy).

## Jak dodać notatkę (Windows)

W PowerShell:

```powershell
cd "D:\Projekt SDA\KSIAZKA_QA"
.\skrypty\dodaj-notatke.ps1 -Tytul "TESTron — panel dziala" -Tresc "HTTP 200 dla dbcraftmode.pl, Supabase + API lokalne."
```

Bez parametrów — skrypt zapyta Cię o tytuł i treść.

## Pliki

| Plik | Zawartość |
|------|-----------|
| `dziennik_2026.md` | Wszystkie wpisy chronologicznie (dołączane automatycznie) |
| `szablon_wpisu.md` | Wzór jednego wpisu |

## Powiązanie z książką

Notatki z tego folderu są źródłem do:
- aktualizacji rozdziałów (`rozdzialy/05_testron.md` itd.),
- pliku `STRESZCZENIE_LIST_MOTYWACYJNY.md` (kilka zdań do CV).

*Nie musisz pisać eseju — 3–5 zdań wystarczy.*
